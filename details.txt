=========================================================
SMARTANNOTATE AI
Active Learning Auto Annotation Platform
=========================================================

DESKRIPSI
---------------------------------------------------------
SmartAnnotate AI adalah platform anotasi berbasis AI yang
memungkinkan pengguna hanya melakukan anotasi awal pada
5-10 gambar, kemudian sistem secara otomatis melakukan
segmentasi, object detection, dan auto-labeling terhadap
gambar lainnya menggunakan model YOLO yang terus dilatih
secara bertahap (incremental training).

Platform menerapkan Human-in-the-Loop Active Learning
sehingga setiap koreksi pengguna akan menjadi data baru
untuk meningkatkan akurasi model secara berkelanjutan.

=========================================================
MASALAH YANG DISELESAIKAN
=========================================================

- Proses anotasi dataset sangat lama
- Membutuhkan banyak tenaga annotator
- Training model memerlukan ribuan label
- Banyak pekerjaan repetitif
- Sulit mengelola dataset besar

=========================================================
SOLUSI
=========================================================

1. User anotasi 5-10 gambar pertama
2. YOLO melakukan training awal
3. AI menghasilkan anotasi otomatis
4. User hanya review dan koreksi
5. Sistem retraining otomatis
6. Model semakin akurat
7. Dataset selesai jauh lebih cepat

=========================================================
FILE NAMING SYSTEM
=========================================================

Nama file anotasi selalu mengikuti nama file gambar.

Contoh:

images/
├── mobil_001.jpg
├── mobil_002.jpg
├── mobil_003.jpg

labels/
├── mobil_001.txt
├── mobil_002.txt
├── mobil_003.txt

Atau COCO:

images/
├── mobil_001.jpg
├── mobil_002.jpg

annotations/
├── mobil_001.json
├── mobil_002.json

Atau Segmentation:

images/
├── drone_001.png

masks/
├── drone_001.png

Dengan demikian setiap file anotasi memiliki nama yang
identik dengan gambar sumber.
